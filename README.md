# khoa-lib-s100fc
Visual C++ IHO S-100 Feature Catalogue library

## Major features
- Read a feature catalogue. 
- Support S-100 4.0
- Retreive detailed information of a feature catalogue.

## Requiremtns
- Visual Studio 2017+
- Visual C++ 32bit
- Windows 7+
- [pugixml](https://pugixml.org/)
